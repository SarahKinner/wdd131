WDD 131.01

Overview

Purpose

Pupose of this site is to store characters I have along with any info about them so if I forget about it I can look at it and remember what importance they have.

Audience

The audience is for my friends that are intereseted in the characters I have made and want to learn more about them. Since there's a lot of info I have on them it'll be easier for them to look through the website instead of me telling them.

Dynamic elements

I'm going to use Javascript in the website by adding a search button which will bring characters up if their name or tags are a match to what the user searched up and an organizer like alphabtecial if the user wants to look through each character in a certain way.
